# Redesign Scope: Converge-Inspired Sapiens Health Pass

This redesign pass will rebuild the homepage around a wider desktop composition so the site fills large screens gracefully and avoids the current boxed, cramped feeling. The visual system should use fewer nested panels and instead rely on larger hero zones, spacious section layouts, and more intentional graphic emphasis.

The hero should follow a two-column presentation similar in structure to the reference direction: a large statement block, concise supporting copy, and clear CTAs paired with a large visual field. The visual field should feature healthcare-specific graphics rather than abstract UI alone. Candidate assets include CT-style imagery, pathology-like textures, waveform or monitor traces, note fragments, EHR-style cards, and AI-generated multimodal fusion objects.

The About or framework section should be simplified into visually understandable product modules. It should show three clean stages: diverse clinical inputs, the MEDMIX engine at the center with Sapiens Health branding, and output tasks on the right. Each stage should be supported by stronger imagery and iconography rather than dense research-style labels.

The redesign needs a balanced set of assets. At minimum, the page should include a hero visual panel, several medical-image cards or modules, branded iconography around the MEDMIX engine, and lighter motion or hover animation that makes the graphics feel alive without overwhelming the page.

## Required asset categories

| Category                      | Purpose                                                                            |
| ----------------------------- | ---------------------------------------------------------------------------------- |
| Hero brand visual             | Establish a premium first impression with Sapiens Health colors and deep-tech mood |
| Clinical imaging visuals      | Show CT, scan, pathology, and monitoring-style inputs in a recognizable way        |
| Product framework graphics    | Present the MEDMIX flow as a clean platform story rather than a research diagram   |
| Small icons and visual labels | Improve readability and help users understand each module quickly                  |
| Motion treatments             | Add subtle movement to image cards, background shapes, and engine activity         |

## Implementation priorities

| Priority | Goal                                                                      |
| -------- | ------------------------------------------------------------------------- |
| 1        | Fix full-screen desktop scaling and section width usage                   |
| 2        | Replace text-heavy blocks with larger visual modules                      |

| 5        | Validate responsive behavior after the visual rebuild                     |
