# Sapiens Health Hero Animation Brief

## Goal

Create a **6-second hero animation** for the Sapiens Health homepage that can replace the current static hero image while preserving the site's existing visual language: a dark purple, clinical, minimal, high-fidelity healthcare AI scene.

## Narrative

The animation should communicate the same three-step product story already present on the page.

| Time      | Story beat     | Visual action                                                                                                                                                                                                                              |
| --------- | -------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 0.0s–1.5s | **Connect**    | A primary care doctor stands at a workstation beside a patient-monitor-style display. Data streams in from EMR, inbox, referral portal, and lab source cards. Thin luminous lines connect those sources into a central clinical workspace. |
| 1.5s–3.5s | **Prioritize** | The incoming information condenses into a clean vertical task list. Tasks reorder themselves automatically by urgency and deadline. One or two rows softly glow to indicate prioritization.                                                |
| 3.5s–6.0s | **Execute**    | An AI assistant layer moves across the interface, drafting follow-up notes, referral packets, or coding suggestions. Short drafting strokes, completion states, and subtle approval cues appear while the doctor remains in control.       |

## Visual Style

The motion should feel **calm, precise, and premium**, not playful. The scene should match the current website art direction and its existing hero graphic.

| Attribute   | Direction                                                                                                                                     |
| ----------- | --------------------------------------------------------------------------------------------------------------------------------------------- |
| Palette     | Deep black-purple background with low-contrast indigo gradients, muted slate devices, and a restrained light-purple accent close to `#c8b7ff` |
| Environment | Clinical workstation or care-operations monitor scene rather than a generic office; clean, minimal, softly lit                                |
| Composition | Wide hero-friendly composition with the main action centered and enough negative space near edges for responsive cropping                     |
| Motion      | Smooth gliding transitions, subtle floating panels, clean line-drawing animation, no exaggerated camera moves                                 |
| Tone        | Quiet confidence, operational clarity, AI-native workflow support                                                                             |

## Required Elements

The final animation should include all of the following.

| Element             | Requirement                                                                                    |
| ------------------- | ---------------------------------------------------------------------------------------------- |
| Doctor              | One clinician or care coordinator visibly working at a monitor                                 |
| External sources    | At least three distinct external data inputs such as EMR, email, referral, or lab portal cards |
| Connection motion   | Visible connective paths or data trails flowing into one unified context                       |
| Task prioritization | A ranked task list that populates and reorders itself                                          |
| Execution layer     | Drafting or workflow completion behavior driven by an AI assistant overlay                     |
| Brand fit           | Visual consistency with the site's current dark monochrome purple aesthetic                    |

## Prompt Draft

Create a **6-second cinematic product concept animation** for a healthcare AI startup website hero section. Show a **primary care doctor** in a dark, premium clinical workstation scene, working beside a large monitor. The environment should feel minimal, polished, and high-fidelity, with a deep black-purple background and restrained light-purple accents matching a modern healthcare AI brand.

In the first part of the animation, show **external data sources** floating in from the sides as elegant interface cards labeled like EMR, email, referrals, and labs. Thin luminous connection lines should route these fragmented inputs into one unified care workspace. In the middle, the data should transform into a **prioritized task list** that automatically populates and reorders by urgency and deadline. In the final part, show an **AI assistant execution layer** moving fluidly through the interface, auto-drafting follow-up notes, referral packets, and coding suggestions with subtle drafting strokes and completion indicators, while the doctor stays in control.

Use a **clean isometric or semi-isometric product-illustration style** similar to a premium healthcare software hero graphic. Motion should be smooth, calm, and precise, with floating cards, soft glow lines, and gentle interface transitions. Avoid bright colors, cartoon aesthetics, busy scenes, extra people, dramatic camera swings, or loud effects. Keep the composition wide and suitable for a startup landing page hero image.
