# Converge Bio design notes for Sapiens Health refinement

The reference site uses a **dark editorial product aesthetic** with very large sans-serif headlines, restrained body copy, and wide negative space. The typography feels straight, neutral, and high-contrast rather than ornamental. Headlines are bold and oversized, while supporting copy is shorter and set in a calmer, smaller sans-serif.

The homepage composition relies on **large left-aligned text blocks**, strong asymmetry, and generous spacing between modules. Sections feel premium because the page avoids clutter and lets a few elements dominate each viewport.

Visually, the site pairs clean typography with **simple framed image blocks**. The cards are uniform, softly rounded, and consistent in size. Images are allowed to be the main content inside each card, with minimal extra decoration. This is the right direction for the MedMIX input area: fewer words, more direct visual recognition, and consistent tile sizing.

For Sapiens Health, the adaptation should be:

1. Replace the current futuristic icon look with **modality-realistic image tiles** that still share one framing system.
2. Reduce overt thematic language such as "Digital Health" in the hero and instead use a more product-like phrase such as **intelligence layer for incomplete clinical data**.
3. Keep the page dark, but simplify the tile section so it feels like a product showcase rather than a concept graphic.
